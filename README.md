# CSS Selectors Demonstration

## Overview
This project demonstrates the use of various CSS selectors and styling techniques in HTML. It showcases how classes, IDs, element selectors, and inline styles can be used to customize the appearance of web page elements.

## Features

- Class Selector (.head)
  - Applied to multiple header elements.
  - Changes the text color to dark blue.

- ID Selector (#bglime)
  - Applied to a specific header.
  - Sets the background color to green-yellow.

- ID Selector (#head)
  - Applied to a word inside an <h2> tag using a <span>.
  - Changes the text color of the word "head" to red.

- Element Selector (p)
  - Applies a cursive font family to all paragraphs.

- ID Selector (#para)
  - Styles a specific paragraph with:
    - Green text color
    - Larger font size

- Inline Styling
  - Sets the webpage background color.
  - Styles the <mark> element to display the word "paragraph" in black with a larger font size.

## Technologies Used

- HTML5
- CSS3

## Learning Outcomes

Through this project, the following concepts are demonstrated:

1. Class Selectors
2. ID Selectors
3. Element Selectors
4. Inline CSS
5. External Stylesheets
6. Text Formatting
7. Background Styling
